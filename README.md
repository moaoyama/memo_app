# Memo App

シンプルにメモを記録・整理できる Web アプリです。

「すぐ書けて、後から探せる」ことを重視して作成しました。

## アプリ概要

- メモの作成・編集・削除ができる
- タグ（カテゴリ）による分類が可能
- キーワード検索・タグ検索に対応
- 誤操作を防ぐ最低限の UX を実装

## 主な機能

- メモ一覧表示（ページネーション対応）
- メモの新規作成・編集・削除
- Markdown記法に対応
- キーワード検索（タイトル・本文）
- タグ（カテゴリ）による絞り込み
- 空状態（Empty State）の表示
- フラッシュメッセージ表示
- 削除時の確認ダイアログ（誤操作防止）
- 作成後30分で自動削除されるメモ機能（チェックで有効化）

## 使用技術

- Ruby on Rails/Ruby
- HTML / CSS
- JavaScript
- Pagy （ページネーション）

## 開発環境

- Ruby 3.3.0
- Rails 7.1.6
- SQLite3（sqlite3 gem）

## 工夫した点

- Empty State を明示

  メモが存在しない場合・検索結果が0件の場合で表示文言を分け、ユーザーが次に何をすればよいか分かるようにしました。

- 誤操作防止を最低限実装

  削除時に確認ダイアログを表示し、編集画面から安全に戻れる導線を用意しました。

- 機能を詰め込みすぎない設計

  「メモを書く・探す」という基本体験を壊さないことを優先しています。

## セットアップ方法

```bash
git clone <https://github.com/moaoyama/memo_app.git>
cd memo_app
bundle install
rails db:migrate
rails s
```

ブラウザで [http://localhost:3000](http://localhost:3000) にアクセスしてください。
