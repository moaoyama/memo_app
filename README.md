# Memo App

シンプルにメモを記録・整理できる Web アプリです。
「すぐ書けて、後から探せる」ことを重視して作成しました。

## アプリ概要

- メモの作成・編集・削除ができる
- タグ（カテゴリ）による分類が可能
- キーワード検索・タグ検索に対応
- 誤操作を防ぐ最低限の UX を実装

## 主な機能

- メモ一覧表示（ページネーション対応）
- メモの新規作成・編集・削除
- Markdown記法に対応
- キーワード検索（タイトル・本文）
- タグ（カテゴリ）による絞り込み
- 空状態（Empty State）の表示
- フラッシュメッセージ表示
- 削除時の確認ダイアログ（誤操作防止）
- チェックを入れると30分後に自動削除されるメモ機能
